# Miliband-Components

Miliband Components are components which build up a broader page. These are designed to be used for the Miliband Theme. This project was made for [The Social Review](https://www.thesocialreview.co.uk/) This documentation is built using storybook, it provides a modular way to test and implement web components.

## Install
First of all, clone the [Github Repo](https://github.com/JacobWeinbren/miliband-components). Then you can change directory into it.

To install dependencies and build the component library
```bash
    yarn && yarn run build
```

For the storybook
```bash
    yarn run storybook
```
