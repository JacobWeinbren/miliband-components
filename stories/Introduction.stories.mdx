# Miliband-Components

Miliband Components are components which build up a broader page. These are designed to be used for the Miliband Theme. This project was made for <Link href="https://www.thesocialreview.co.uk/">The Social Review</Link>

## Install
Clone the <Link href="https://github.com/JacobWeinbren/miliband-components">repo</Link>.

Then run
<ClipboardCode code="yarn && yarn run build"/>

For the storybook, run 
<ClipboardCode code="yarn run storybook"/>.
