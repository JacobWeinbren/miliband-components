# Miliband-Components

Miliband Components are components which build up a broader page. These are designed to be used for the Miliband Theme. This project was made for [The Social Review](https://www.thesocialreview.co.uk/)

## Build
Clone the <Link href="https://github.com/JacobWeinbren/miliband-components">repo</Link>, cd into the directory, then run `yarn install && yarn run build`

For the storybook, run `yarn run storybook`.

## Further

You can find out more details on the [Github Repo](https://github.com/JacobWeinbren/miliband-components)
